# 🫒 سلافة - متجر زيت الزيتون الإلكتروني

موقع متكامل لبيع زيت الزيتون مبني بـ Python (Flask) + SQLite

## المتطلبات
- Python 3.8+
- Flask

## تشغيل المشروع

```bash
# 1. تثبيت المكتبات
pip install -r requirements.txt

# 2. تشغيل السيرفر
python app.py

# 3. افتح المتصفح على
http://localhost:5000
```

## الصفحات
- `/` — الصفحة الرئيسية
- `/products` — جميع المنتجات مع الفلترة والبحث
- `/product/<id>` — تفاصيل المنتج والتقييمات
- `/cart` — سلة الشراء
- `/checkout` — صفحة الدفع الآمن بالفيزا
- `/about` — عن المتجر

## قاعدة البيانات (SQLite)
- `products` - المنتجات
- `orders` - الطلبات
- `order_items` - تفاصيل الطلبات
- `cart` - سلة الشراء
- `reviews` - تقييمات العملاء
